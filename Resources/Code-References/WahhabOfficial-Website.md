# WahhabOfficial Website — Code Reference

> Summary of the WahhabOfficial website repository. A git-tracked web project for [[Wahhab]]'s official artist website.

---

## Overview

The WahhabOfficial directory is a full website repository (with `.git` history) containing the official artist website for Wahhab / DBLU. It includes standard web assets (HTML, CSS, JavaScript, JSON), fonts, images, and SVGs, as well as non-code content directories for music, strategy documents, and creative assets.

## Tech Stack

- HTML / CSS / JavaScript (static site)
- Custom fonts (WOFF/TTF)
- SVG assets and imagery

## Non-Code Content (Also in This Repo)

The repository also houses several knowledge/strategy directories that have been migrated into the vault separately:

- `Series/` → [[Series-Notes]] (animated series creative bible)
- `Strategy/Distribution Plan/` → [[Souq-el-So7ab-Distribution-Plan]]
- `Strategy/Marketing/` → [[Souq-el-So7ab-Marketing-Strategy]] + [[Souq-el-So7ab-Content-Execution-Plan]]
- `Wahhab/Wahhabs Brand Brochure.pdf` → [[Wahhab-Brand-Brochure]]
- `Wahhab/Wahhab's Strategic Market Analysis.pdf` → [[AI-Customer-Service-Market-Analysis]]
- `Wahhab/Music/Singles/` — Audio files (WAV/MP3) for released and unreleased tracks
- `Wahhab/Merch Designs/` — PSD files and design assets for bucket hats and apparel

## Music Files Present

Singles directories include audio for: I Have a Dream, It Just Me, Cookout, Hide and Seek, Le3ba, No Sleep (ft GoOFs), Hashteg (هاشتِك), Kenz, A Man's Tear, Adoor 3aleek, 3aks 3ekas.
