# CreativeSpaces — Brand Color Exploration

> Interactive HTML page exploring four brand color direction options for CreativeSpaces. Each option includes full color systems (primary, accent, surface, text, border, gradient tokens) with UI component previews.

---

## Color Options Explored

### Option A: Forest Elite
Dark green direction. Deep forest base with bright emerald accents.
- Primary: `#0D2818` (deep forest)
- Accent: `#2ECC71` (bright emerald)
- Surface: `#0A1F14`
- Card: `#112A1C`
- Text: `#E8F5E9`
- Gradient: deep forest → mid green → dark teal

### Option B: Emerald Noir
Deeper, more mysterious green palette. Near-black base with neon green accents.
- Primary: `#071A0F` (near-black green)
- Accent: `#00E676` (neon emerald)
- Surface: `#050F09`
- Card: `#0A1A10`
- Text: `#E0F2E0`
- Gradient: very dark → deep green → dark

### Option C: Sage Luxury
Warmer, earthier green direction. Olive-green tones with gold-green accents.
- Primary: `#1A2418` (olive-dark)
- Primary-light: `#2D3E2A`
- Accent: warm sage-gold tones
- More approachable and organic feel than Options A/B

### Option D: (Fourth variation)
Additional direction explored in the HTML file for comparison.

---

## Format

The source is a self-contained HTML page with CSS custom properties (design tokens) and live-rendered UI component previews (cards, buttons, gradients, typography) for each color option. Font: Inter. All four options use dark-mode-first design with green as the primary brand color family.
